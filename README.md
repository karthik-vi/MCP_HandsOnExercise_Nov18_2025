# MCP Server Demo - Data Intensive Computing

## Overview
In this hands-on workshop, we'll build a Model Context Protocol (MCP) server from scratch, following the official MCP tutorial from [modelcontextprotocol.io](https://modelcontextprotocol.io/docs/develop/build-server).

## What You'll Learn
- Building MCP servers with tools, resources, and prompts
- Implementing data analysis workflows through MCP
- Creating structured data interfaces for AI systems
- Understanding JSON-RPC communication protocols

## Setup Instructions

### 1. Create GitHub Codespace
1. Fork this repository
2. Click "Code" → "Codespaces" → "Create codespace"
3. Wait for environment to initialize

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. We'll Build Together
- MCP server structure
- Data analysis tools
- Resource management
- Workflow prompts
